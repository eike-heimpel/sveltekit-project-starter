# Sveltekit & Supabase starter project

- Sveltekit
- Tailwind 
- Daisy UI
- Supabase connected / access to supabase via locals
- Auth implemented via supabase (/auth route)

## Get Started

1. Rename .env_example to .env 
2. add your supabase info in the .env file
3. `npm install`
4. `npm run dev`
